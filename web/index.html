How do I get an .evtx file?
  1. On the Windows machine, open Event Viewer (press Win+R, type eventvwr).
  2. Right-click a log — usually Windows Logs → System or Application — and choose Save All Events As…
  3. Save it as an .evtx file and open it here.

Prefer a desktop app? The Windows version of this tool also renders the full event messages.
